A-系统安装方法：
步骤1、配置conf/main.conf.php中的数据库信息（不需要自己建立数据库）
步骤2、访问install/install.php。
经过以上步骤，即可搭建一个全新的纯净系统。
步骤3、为保证数据安全性，在创建数据库之后务必删除install文件夹。

B-多套系统共同运行方案：
实现方案：为每套系统建立一个数据库，修改conf中的配置信息，即可实现多套系统共同运行。
